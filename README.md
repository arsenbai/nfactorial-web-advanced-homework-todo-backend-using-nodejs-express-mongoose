# nFactorial School Web Advanced Howework

### Deadline: May-7, 2023
### Howework name: To-Do-List Backend

### Technologies used:
- NodeJS/Express
- NodeJS/Mongoose

## Prerequisites
You need to create database called 'ToDoDB' in MongoDB before running this backend code

## Installation and script running cmd

### Install using package.json dependencies
`npm install`

### Run server
`npm run dev`
